# Cursor Website Clone

This project is a static recreation of the Cursor homepage using only HTML and CSS.

## Sections Implemented
- Navigation Bar
- Hero Section
- Trusted By Section
- Feature Sections
- Feature Cards
- Testimonials
- Final Call To Action
- Footer

## Technologies Used
- HTML5
- CSS3
- Google Fonts (Inter)

## Color Palette
- Background: #0d0d0d
- Cards: #111 / #1a1a1a
- Primary Text: #ffffff
- Secondary Text: #a0a0a0

## How To Run
Open index.html in your browser.
